# レビュー機能追加の完了報告

ユーザーの要望に基づき、各ギャンブル（パチンコ、麻雀、競馬、競艇、オート、競輪）に対するレビュー（評価）機能を追加しました。

## 追加された機能の概要

### 1. レビュー対象のデータモデルとFirestore保存
新たに以下のFirestoreコレクションに対するデータモデルと、それを操作する `ReviewService` を作成しました。
- **パチンコ台** (`pachinko_machine_reviews`): 総合評価、演出、出玉力、カスタム、音楽、メモ
- **パチンコ店舗** (`pachinko_place_reviews`): 総合評価、回転数、雰囲気、店員、メモ
- **麻雀店舗** (`mahjong_place_reviews`): 総合評価、アクセス、値段、雰囲気、店員、メモ
- **競馬 馬** (`horse_reviews`): 総合評価、スピード、スタミナ、パワー、メモ
- **競馬 騎手** (`jockey_reviews`): 総合評価、技術、勝負強さ、ペース配分、メモ
- **公営競技(競輪/競艇/オート) 選手** (`racer_reviews`): 総合評価、スピード、技術、メンタル、メモ

### 2. UIの追加
- **ホーム画面 (`HomeScreen`)**: AppBarを設置し、右上に「⭐︎ (レビュー管理)」ボタンを追加しました。
- **レビューメイン画面 (`ReviewMainScreen`)**: 上部にTabBarを配置し、各ギャンブルのタブを切り替えられるようにしました。
- **各タブの画面**:
  - **パチンコタブ**: 既存の台(`MachineType`)、店舗(`Place`)データを読み込み、リスト表示します。各項目をタップしてレビュー情報を入力・保存できます。
  - **麻雀タブ**: 既存の店舗(`Place`)データを読み込み、リスト表示・レビュー編集が可能です。
  - **競馬タブ**: 馬と騎手を切り替えて表示。右下の「＋」ボタンから新しい馬・騎手を追加して評価できます。
  - **競輪・競艇・オートタブ**: カテゴリに応じた選手のリストを表示。右下の「＋」ボタンから新しい選手を追加して評価できます。
- **共通機能**:
  - **評価入力**: 1〜5の星をタップして入力できる `StarRatingInput` ウィジェットを作成。
  - **絞り込みと並べ替え**: 各画面右上のフィルターアイコンから、「最低総合評価の絞り込み」および「名前・総合評価での並べ替え（昇順・降順）」ができるBottomSheetを実装しました。

## 確認事項
- `dart analyze` による静的解析を実施し、今回追加した機能がビルドエラーなく動作する状態であることを確認しました（一部の Info 通知は、既存のFlutter/Dart仕様の変更推奨によるものです）。
- パチンコと麻雀については、既存のマスタ追加機能を使って追加された台や店舗が自動的にリストに表示されます。
- パチンコ以外については、レビュー画面内で直接要素を追加・削除できるよう実装しています。
