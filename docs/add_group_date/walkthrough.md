# 「日付」グループオプションおよび横画面グラフの実装完了

ご要望通り、詳細タブのグループオプションに「日付」での集計機能を追加し、グラフを横画面で表示できるよう実装しました。

## 変更内容

### 1. 「日付」グループオプションの追加 (`GroupingHelper`)
- 「詳細」モードのグループオプションに「日付」を追加しました。
- データの集計単位を制御するロジックを新設しました。
  - **年間**: 各履歴を「2023年」や「2024年」といった単位でグループ化します。
  - **月間**: 各履歴を「2023/4」や「2023/5」といった月単位でグループ化します。
  - **週間**: 各履歴が属する週の初め（月曜）を見つけ「2023/4/3〜」といった週単位でグループ化します。
  - **カスタム**: 「2023/4/3」のように日単位でグループ化します。
  - 日付は金額順ではなく、時系列（古い順から新しい順）になるように並び替えを行っています。グラフ上で時系列の推移がわかりやすくなります。

### 2. 期間選択のためのスタイリッシュなラジオボタン (`CategoryView`)
- グループ化で「日付」を選択した際に、ドロップダウンメニューの下におしゃれなデザインの `SegmentedButton`（週間・月間・年間・カスタム）を表示するように変更しました。
- デフォルトは「年間」が選択されています。
- 「カスタム」を選択すると自動でカレンダー形式の期間選択ダイアログが開き、範囲指定が可能です。
- 「カスタム」で範囲指定された場合は対象データが指定期間の中に収まっているものだけにフィルタリング（絞り込み）され、その絞り込まれた結果だけで合計金額の計算やリスト表示、グラフ生成が行われます。

### 3. グラフの横画面表示 (`ChartScreen`)
- グラフを大きく見やすくするため、グラフ画面(`ChartScreen`)を開いた瞬間にスマートフォンの画面が強制的に横向き(`Landscape`モード)になるように実装を行いました。
- グラフ画面を閉じて前のリスト画面に戻った際には、元の縦向きに自動で戻るように処理を追加しています。

## 使い方
1. 個別データの「詳細」タブを表示。
2. グループから**日付**を選択します。
3. すぐ下に出現する **[週間 | 月間 | 年間 | カスタム]** のボタンから見たい単位を選びます。
  - 過去全てのデータが、ここで選んだ単位ごとにまとめられてリスト表示されます。（カスタムは選択した期間内のみ）
4. **グラフマーク**のボタンを押すと、画面が横向きになり、時系列で並んだ収支グラフが表示されます。
