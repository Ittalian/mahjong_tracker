# 詳細表示の実装完了

各競技の結果画面に「基本表示」と「詳細表示」を切り替える機能を追加しました。詳細表示では、特定のプロパティ（タイプ、レートなど）ごとに結果をグループ化して集計表示できます。さらに、集計されたグループをタップすることで、そのグループに属する記録の一覧を確認できる「ドリルダウン機能」を実装しました。

## 変更内容

### UI

- **基本/詳細 切り替え**: 画面上部（合計収支の下）に「基本」「詳細」の切り替えボタンを追加しました。
- **グループ化選択**: 詳細モード時には、グループ化の基準となるプロパティ（「タイプ」「レート」「メンバー」など）を選択するドロップダウンが表示されます。
- **集計リスト**: 選択したプロパティごとにグループ化された合計収支がリスト表示されます。
- **ドリルダウン**: 集計リストの項目をタップすると、そのグループに該当する記録のみがフィルタリングされて表示されます。上部の「< 戻る」ボタンで集計リストに戻れます。

### ロジック

- `GroupingHelper` クラスを作成し、各競技（麻雀、競馬、パチンコなど）のデータ構造に合わせて集計・フィルタリングを行うロジックを集約しました。
- 麻雀やパチンコの `member` リストのように、1つの結果が複数の要素に関係する場合、それぞれの要素（メンバー）に対して金額を加算するロジックを実装しました（メンバーがいない場合は「ソロ」として集計）。

## 検証結果

### 手動確認手順
1. アプリを起動し、任意の競技タブを選択します。
2. 「詳細」ボタンをタップします。
3. ドロップダウンからグループ化したい項目を選択します（例：「メンバー」）。
4. 項目ごとに集計された収支が表示されることを確認します。
5. 集計された項目（例：「Aさん」）をタップすると、Aさんが含まれる記録一覧が表示されることを確認します。
6. 「< (グループ名) の詳細」の戻るボタンで集計画面に戻ることを確認します。
