## アプリの概要

FlutterとFirebase (Firestore) を使用した、個人用の麻雀収支管理アプリケーションです。
日々の対局結果（日付、金額、メモ）をクラウドに保存し、リアルタイムで合計収支を確認できます。

### 主な機能
- 収支の一覧表示と合計金額の自動計算
- 収支の追加・編集・削除
- ダークモード対応（システム設定に依存）

## 実行方法

Firebaseの設定（`walkthrough.md` 参照）完了後、以下のコマンドで実行します。

```bash
flutter run
```
