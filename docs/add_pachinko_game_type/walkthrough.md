# パチンコ競技追加のウォークスルー

新しい競技「パチンコ」をアプリに追加しました。

## 実施した変更

### 1. モデルの作成
- `PachinkoResult` モデルを作成しました。入力項目は麻雀と同じ「日付」「金額」「メモ」のみで、賭け方はありません。

### 2. サービス層の更新
- `FirestoreService` にパチンコ用のCRUDメソッドを追加しました。
  - `addPachinkoResult`
  - `getPachinkoResults`
  - `updatePachinkoResult`
  - `deletePachinkoResult`

### 3. UIの更新
- **ホーム画面 (`HomeScreen`)**:
  - カテゴリタブに「パチンコ」を追加しました。
  - 一覧表示と削除機能に対応しました。
- **編集画面 (`EditScreen`)**:
  - パチンコの収支追加・編集に対応しました。
  - パチンコ選択時は「賭け方」入力欄が表示されないように制御しました。
- **サマリー画面 (`SummaryScreen`)**:
  - 総合計収支とカテゴリ別収支にパチンコを含めるようにしました。

## 検証結果

以下の動作が正常に行われることを（コード上で）確認しました。
- パチンコ画面へのナビゲーション
- 収支の追加、編集、削除
- 総合計への反映
