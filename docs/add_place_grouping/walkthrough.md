# UI調整の実装が完了しました

ご要望いただいた3点のUI調整が完了しました。

## 1. 詳細画面での場所ごとの絞り込み
- パチンコ以外の競技（麻雀、競馬、競艇、オートレース、競輪）の詳細画面（CategoryView）でも、グループ一覧から「場所」を選択できるようになりました。
- これにより、すべての競技において場所ごとの収支の集計・絞り込み表示が可能になっています。

## 2. 詳細グラフの縦向き化
- グラフ画面（`ChartScreen`）を開いた際に、画面が強制的に横向き（Landscape）になる仕様を廃止しました。
- スマホを縦に持ったままの状態でグラフ画面が表示されるようになり、縦画面内でグラフを横スクロールして確認できます。

## 3. セレクトとキーボードの被り修正
- 場所や台の種類を入力するセレクト（サジェストリスト）を展開した際、リストの最大高さをキーボードの高さを考慮して動的に計算するようにしました。
- 画面サイズからキーボードの領域を差し引いた空きスペース（の40%程度）に収まるようにサジェストのボックスが調整されるため、キーボードの後ろに隠れて操作できなくなる問題が解消されました。

## 確認のお願い
アプリをリロード（再ビルド）していただき、以下の点をご確認ください。
- 詳細画面で「グループ：場所」が選べるか
- グラフを開いた際に画面が縦向きのまま表示されるか
- セレクトを開き、ソフトウェアキーボードを出したときにサジェストがキーボードに隠れず正しく表示されるか
